# Sheffield Interview Speedrun
A text based game using object oriented programming practices made with HTML and Javascript

## Instructions:
### 1. Enter username

### 2. Press start

### 3. Type one of the listed directions to move buildings (north, east, south, west)

### 4. Press enter

### 5. Explore the rooms looking for clues of where your interview might be

### 6. Find the interview before the time runs out. Each move takes 5 mintes!
